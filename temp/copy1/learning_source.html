ตัวอย่างวิดีโอ: พื้นฐานการใช้ printf

พื้นฐานการใช้ printf (2:05)

เรียนรู้วิธีการใช้ฟังก์ชัน printf เพื่อแสดงผลข้อความและตัวแปรในภาษา C

โค้ดในวิดีโอ:

#include <stdio.h>

int main() {
    int age = 25;
    float height = 175.5;
    char name[] = "สมชาย";
    
    printf("สวัสดี, ฉันชื่อ %s\n", name);
    printf("ฉันอายุ %d ปี และสูง %.1f ซม.\n", age, height);
    
    return 0;
}

คำอธิบายสำคัญ:

  • %s ใช้สำหรับแสดงข้อความ (string)
  • %d ใช้สำหรับแสดงตัวเลขจำนวนเต็ม (integer)
  • %.1f ใช้สำหรับแสดงทศนิยม 1 ตำแหน่ง
  • \n คือการขึ้นบรรทัดใหม่

ตัวอย่างวิดีโอ: การใช้ if-else

การใช้ if-else (1:45)

เรียนรู้วิธีการใช้คำสั่งเงื่อนไข if-else เพื่อควบคุมการทำงานของโปรแกรม

โค้ดในวิดีโอ:

#include <stdio.h>

int main() {
    int score;
    
    printf("กรุณาป้อนคะแนนของคุณ: ");
    scanf("%d", &score);
    
    if (score >= 80) {
        printf("เกรด A\n");
    } else if (score >= 70) {
        printf("เกรด B\n");
    } else if (score >= 60) {
        printf("เกรด C\n");
    } else if (score >= 50) {
        printf("เกรด D\n");
    } else {
        printf("เกรด F\n");
    }
    
    return 0;
}

คำอธิบายสำคัญ:

  • if-else ใช้ตรวจสอบเงื่อนไขและทำงานตามเงื่อนไขที่เป็นจริง
  • scanf ใช้รับข้อมูลจากผู้ใช้
  • เครื่องหมาย >= หมายถึง "มากกว่าหรือเท่ากับ"
  • เงื่อนไขจะถูกตรวจสอบจากบนลงล่างตามลำดับ
